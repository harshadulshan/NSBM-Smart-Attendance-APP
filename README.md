## 🎯 What is This?

A **production-ready AI Desktop Attendance System** built specifically for **NSBM Green University**. Uses real-time **Face Recognition** to automatically identify and mark student attendance via webcam.

No more manual roll calls. Just show your face! 🤖

---

## ✨ Features

| Feature | Description |
|---|---|
| 🔒 **Secure Login** | Multi-role access (Admin/Lecturer/Student) |
| 📸 **Face Registration** | Register students with name, ID, course & batch |
| 🤖 **AI Recognition** | Real-time face detection & attendance marking |
| 🔴 **Late Detection** | Automatically flags late arrivals |
| ⚠️ **Unknown Alerts** | Detects and logs unregistered faces |
| 📊 **Reports** | Filter by date, course and batch |
| 📅 **Timetable** | Auto-detect current running session |
| 🔍 **Duplicate Check** | Prevents double marking per session |
| 📥 **Excel Export** | One-click attendance report download |
| 📧 **Email Reports** | Auto send daily attendance summary |
| 🎨 **Modern UI** | Glassmorphism design with NSBM colors |
| 📦 **Packaged** | Runs as Windows .exe — no Python needed |
